## Part 1: Knowledge Prerequisites

### Tier 1: Essential (Must Know Before Bootcamp)

#### Molecular Biology Fundamentals
- **The Central Dogma**: DNA → RNA → Protein flow
- **DNA Structure**: Double helix, base pairing (A-T, G-C), 5' to 3' directionality
- **Gene Expression**: Transcription, translation basics
- **Proteins**: Amino acids, folding, function

> **Longevity Relevance**: Understanding how genes encode proteins is foundational for comprehending gene therapies targeting aging pathways (e.g., telomerase, sirtuins, mTOR).

#### Chemistry Basics
- Organic chemistry concepts (functional groups, bonds)
- Basic biochemistry (enzymes, substrates, catalysis)
- pH, buffers, solutions

### Tier 2: Helpful (Will Accelerate Learning)

#### Genetics
- Mutations (point, frameshift, insertion, deletion)
- Gene regulation (promoters, enhancers, repressors)
- Plasmids and vectors

#### Cell Biology
- Cell structure (prokaryotic vs eukaryotic)
- Cell membrane, organelles
- Cell division basics

#### For Your Software Background
- Basic Python syntax (you'll learn this in Day 3, but pre-knowledge helps)
- Familiarity with Jupyter notebooks/Google Colab
- Command line basics

## Part 4: Longevity Biotech Relevance Deep Dive

### How Each Bootcamp Topic Connects to Aging Research

#### 1. Central Dogma & Gene Expression → Aging Pathways

Key aging genes to know:
- **SIRT1-7** (Sirtuins): NAD+-dependent deacetylases regulating metabolism and stress response
- **mTOR**: Master regulator of growth; inhibition extends lifespan
- **FOXO transcription factors**: Stress resistance and longevity
- **TP53**: Tumor suppressor, cellular senescence trigger
- **TERT**: Telomerase component; telomere maintenance

#### 2. CRISPR & Genome Engineering → Gene Therapy for Aging

Current applications:
- Editing cells to remove senescence markers
- Correcting age-related genetic diseases (progeria)
- Engineering cells to produce therapeutic proteins
- Creating animal models of aging

#### 3. Genetic Circuits → Therapeutic Delivery

Potential applications:
- Circuits that sense aging biomarkers and respond
- Programmable senolytic systems
- Living therapeutics for microbiome engineering

#### 4. Phage Therapy → Microbiome & Infection

Relevance:
- Aging microbiome changes correlate with health decline
- Phages can selectively modulate microbiome
- Alternative to antibiotics in immunocompromised elderly

#### 5. Bioinformatics → Aging Data Analysis

Applications:
- Analyzing transcriptomic data from aging tissues
- Epigenetic clock development
- Drug discovery for senolytics
- Biomarker identification

## Part 6: Study Prompts for Self-Learning

Use these prompts with Claude or ChatGPT to generate study materials:

### Central Dogma & Molecular Biology

**Prompt 1:**
```
Explain the central dogma of molecular biology as if I'm a software engineer.
Use analogies to programming concepts (like compilation, interpretation, APIs).
Then explain how this relates to genetic diseases and gene therapy.
```

**Prompt 2:**
```
Create a quiz with 10 questions about DNA replication, transcription, and
translation. Include questions about the enzymes involved and their functions.
Focus on aspects relevant to understanding CRISPR and gene editing.
```

**Prompt 3:**
```
Explain the difference between eukaryotic and prokaryotic gene expression.
Why does this matter for synthetic biology and using E. coli as a chassis organism?
```

### CRISPR & Genome Engineering

**Prompt 4:**
```
I'm learning CRISPR for the first time. Explain:
1. How Cas9 finds its target
2. How guide RNAs are designed
3. The difference between gene knockout and knock-in
4. Current limitations and how they're being addressed
Include recent advances from 2024-2025.
```

**Prompt 5:**
```
Compare CRISPR-Cas9, ZFNs, and TALENs. Create a table showing their
mechanisms, advantages, disadvantages, and best use cases. Which is most
relevant for therapeutic applications?
```

### Bioinformatics & Python

**Prompt 6:**
```
I'm a backend engineer learning bioinformatics. Create a learning roadmap
that covers:
1. Essential biology concepts I need
2. Key Python libraries (Biopython, etc.)
3. Important databases and how to query them programmatically
4. A small project analyzing a gene related to aging (suggest one)
```

**Prompt 7:**
```
Write a Python tutorial using Biopython to:
1. Fetch a gene sequence from NCBI (use SIRT1 as an example)
2. Translate it to protein sequence
3. Perform a BLAST search
4. Visualize the results
Include explanations of the biology at each step.
```

### Longevity & Aging Biology

**Prompt 8:**
```
Explain the hallmarks of aging (López-Otín et al.) and for each hallmark,
describe:
1. The molecular mechanisms
2. How it contributes to age-related diseases
3. Current therapeutic approaches targeting it
4. Key genes/proteins involved
```

**Prompt 9:**
```
I want to understand senolytics. Explain:
1. What cellular senescence is
2. Why senescent cells accumulate with age
3. How senolytics work (mechanism of action)
4. Current clinical trials and their results
5. Key companies working on this
```

**Prompt 10:**
```
Create a study guide for understanding epigenetic clocks. Cover:
1. What epigenetic modifications are
2. How methylation patterns change with age
3. How clocks are trained/calibrated
4. Differences between Horvath, GrimAge, PhenoAge clocks
5. How these are used in longevity research
```

### Synthetic Biology & Genetic Circuits

**Prompt 11:**
```
Explain genetic circuits using programming analogies. Cover:
1. Promoters as 'function calls'
2. Repressors and activators as 'if/else statements'
3. Feedback loops as 'while loops'
4. Give examples of real genetic circuits and their applications
```

## Part 8: Your Personalized Bioinformatics Learning Path

Given your backend engineering background, here's a tailored progression:

### Phase 1: Biology Foundations (1-2 weeks)
1. Complete Khan Academy's molecular biology section
2. Read "Molecular Biology of the Cell" Chapter 1-6 (Alberts)
3. Use Rosalind problems to reinforce concepts

### Phase 2: Python for Biology (2-3 weeks)
1. Coursera: Biology Meets Programming (UC San Diego)
2. Learn Biopython through their official tutorial
3. Practice with NCBI databases programmatically

### Phase 3: Data Analysis (2-3 weeks)
1. Learn pandas/numpy for biological data
2. Galaxy Training Network tutorials
3. Analyze real datasets (GEO database has aging studies)

### Phase 4: Advanced Applications (Ongoing)
1. Machine learning for biology (scikit-learn)
2. Single-cell analysis (Scanpy)
3. Structural bioinformatics (PDB, AlphaFold)

### Longevity-Specific Bioinformatics Projects

1. **Aging Gene Network Analysis**: Download gene expression data from aging tissues, identify differentially expressed genes
2. **Epigenetic Clock Implementation**: Implement a simple methylation clock predictor
3. **Senescence Signature Detector**: Build a classifier for senescent cell markers
4. **Drug Repurposing Analysis**: Analyze compound databases for potential senolytics
